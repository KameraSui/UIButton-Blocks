## Installation

WJHModernLib is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'WJHModernLib',:git => 'https://gitee.com/kamerasui/WJHModernLib.git'
```
